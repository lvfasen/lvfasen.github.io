## (5)修改配置文件node_cache,node_global

npm config set prefix "D:\program Files\nodejs\node_global"

npm config set cache "D:\program Files\nodejs\node_cache"

## (6)配置淘宝镜像

npm config set registry https://registry.npm.taobao.org

## (7)配置国内cnpm命令

npm install -g cnpm
